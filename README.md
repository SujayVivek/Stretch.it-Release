# Stretch.it

Stretch.it is a small Windows desktop app that runs quietly in the background
and reminds you to take short movement breaks while working.

### Features
- Runs in the system tray
- Simple exercise reminders
- No accounts, no tracking, works offline

### Download
Go to the **Releases** section and download the latest Windows installer.

### Notes
- Windows only
- This is an early v1 release
